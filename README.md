# Sales Territory Capstone Project

This capstone project focused on analyzing sales territory data for a growing bookstore business using SQL and Excel. I walk through my thought process and brainstorming while analyzing different parts of the dataset. I also created charts in Excel to support my recommendations on where to focus sales attention in the next quarter.
